# Tic Tac Toe Game Engine (C)
A console-based Tic Tac Toe game implemented in C that demonstrates core game logic, board state management, and win-condition detection.


## Features
- Player vs Player mode
- Player vs Computer mode
- Board representation using a 2D array
- Win-condition detection for rows, columns, and diagonals
- Input validation and turn-based game loop

## Implementation Highlights
- 2D array-based board representation
- Conditional logic for move validation and turn switching
- Win detection across rows, columns, and diagonals
- Modular game flow for repeated turns until win or draw


## Project Structure
- `TicTacToe.c` — main game logic and board management
- `assets/` — terminal screenshots or demo GIFs
- `README.md` — project documentation
